Apple Website Clone – Week 7 Project

This is a fully responsive Apple website clone built using React + Vite, Tailwind CSS, and a bit of JavaScript inside JSX.
The project focuses on UI structure, component reusability, animations, and clean code architecture.

Tech Stack Used:

React + Vite

Tailwind CSS

JavaScript (inside JSX)

React Icons

Responsive design techniques

Features:

Fully responsive layout

Apple-style navbar

Hero section with text + images

Reusable components

Clean, simple UI inspired by Apple’s design

This project is part of my Week 7 learning series, where I focus on improving frontend skills.
